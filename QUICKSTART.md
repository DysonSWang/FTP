# 海南自贸港政策库 - 快速启动指南

## 🚀 一键启动

```bash
cd /root/.openclaw/workspace/hainan-policy
./start.sh
```

## 🌐 访问地址

启动后访问：**http://localhost:5173**

## ✨ 核心功能

### 1. 查看统计
- 政策总数：180+ 条
- 分类数量：11 类
- 最新政策：2025-10-29
- 国家级政策：实时统计

### 2. 搜索政策
在搜索框输入关键词，如：
- "所得税" - 查找税收相关政策
- "2025" - 查找 2025 年政策
- "人才" - 查找人才相关政策

### 3. 分类筛选
点击分类标签快速筛选：
- 顶层设计
- 税收政策
- 人才政策
- 贸易政策
- 投资政策
- 金融政策
- 运输政策
- 产业政策
- 园区政策
- 营商环境
- 法律法规

### 4. 查看原文
点击政策卡片上的"查看原文 →"链接，直达官方政策文件。

## 🛠️ 故障排查

### 端口被占用
```bash
# 检查端口
lsof -ti:5173 | xargs kill -9

# 重新启动
node server.js
```

### 数据不显示
- 检查浏览器控制台是否有错误
- 刷新页面重试
- 检查 `data/policies-complete.js` 是否存在
